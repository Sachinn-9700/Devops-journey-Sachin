# Simple Backup Script

## What it does
Creates a compressed backup of a specified directory and stores it
in a backup location with date and time.

## Why this is useful
Backup automation is a common operational requirement in DevOps
to ensure data safety and recovery.

## Bash concepts used
- variables
- date command
- directory creation
- tar compression
- exit status check

## How to run
```bash
chmod +x backup.sh
./backup.sh
