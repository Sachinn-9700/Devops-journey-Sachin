# Docker Fundamentals

This folder covers the core fundamentals of Docker from a DevOps engineer’s perspective.

## Why this matters
Docker fundamentals are the base for:
- Dockerfile
- Kubernetes
- CI/CD pipelines
- Cloud-native deployments

Without understanding these concepts, Docker becomes command memorization instead of engineering.

## Topics Covered
- Docker architecture
- Image vs container
- Container lifecycle
- Core Docker commands
- `docker run` deep dive
- Common beginner mistakes

## Outcome
After completing this module, I can:
- Explain how Docker works internally
- Confidently run and manage containers
- Debug common container issues
- Transition smoothly into Dockerfile and Kubernetes
